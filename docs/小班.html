小班教学

小班是比较中庸的，比较标准。

教学方式

  1. 计划人数是3~10人。
  2. 线上线下都可以，线下效果会好很多，但需要来杭州。
  3. 一次课最好是2小时课程 + 6小时辅导，哪怕没有这个条件，至少也要有2小时的专心上课时间。
  4. 很多看书就能懂的内容并没有讲授的必要，因此课上只讲书上学不到的内容，学生必须保质保量完成作业。
  5. 小班课堂不可能像一对一那样高效，因此对课后学生交流的要求很高，学生必须更主动些。
  6. 我会安排足够的学生交流，请积极互相帮助，方可达到足够的学习效率。

招收标准

  1. 有兴趣。
  2. 有时间。
  3. 积极思考。
  4. 认真完成作业。

课程计划

小班教学自然是按照课时收费。
  • 课时参考价格：200￥/小时。
  • 课程计划：
    1. 算法与数据结构，4000￥
    2. 基本数据分析（Excel+SQL），1600￥
    3. 高级数据分析（Python+画图），2400￥；相关算法与数据结构基础，1600￥
    4. 机器学习基础，2400￥；大数据技术、专有领域，具体情况具体分析
    5. 有别的想学的话，详谈。
  • 相关免费服务：
    1. 试听课 + 个人学习计划定制
    2. 后续相关答疑与辅导 + 学习小组
    3. 业内一线资源（前辈咨询、内推等）
  • 以上课程时间均为针对零基础学生的参考，具体根据学生调整，充分个人定制。
