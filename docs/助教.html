助教教学

助教往往是在校学生最需要的，能针对大小疑问指导解决思路、探索方向。
如此，学习效率可以得到大幅度的提高，而且相关视野的扩展也绝对是物超所值的。

教学方式

  1. 目前处于试验阶段，暂不限人数。
  2. 我会设计足够且合适的学习路线指导，当然也支持学生自己大学的课程。
  3. 有问题直接线上提问，采用文字、语音、屏幕共享来交流。
  4. 助教方式只指导、不解决问题，一切问题的最终解决都是靠学生自己的探索和努力的。

招收标准

  1. 有充分的学习时间。
  2. 有基本的自学能力。
  3. 有学习动力，不懒。

课程计划

助教教学采用订阅制，目前处于试验阶段，欢迎大家踊跃参与。
  • 课时参考价格：500￥/月。
    1. 试验阶段，订阅期间可以随时提问，暂不限制交流的次数与总时间。
    2. 试验阶段，会尝试组织一些小组同学交流，形式估计是在线会议。
    3. 试验阶段，还会尝试各种能加强学习效果的方式。
    4. 试验阶段截止日期待定，截止之前会通知每个学生的。
  • 参考课程：
    1. 算法与数据结构
    2. 基本数据分析（Excel+SQL）
    3. 高级数据分析（Python+画图）；相关算法与数据结构基础
    4. 机器学习基础；大数据技术、专有领域，具体情况具体分析
    5. 有别的想学的话，详谈。

报名方式

写一张报名表发给我，联系方式见关于我。 报名表包含如下内容：

  • 姓名、性别、年龄
  • 学校、专业、年级
  • 希望学习的课程
  • 计划付出的学习时间
  • 相关疑问

欢迎大家踊跃参与，我会尽快回复。
