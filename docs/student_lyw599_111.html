辅导计划

按照1年达到合格毕业生的进度，须得达到以下目标：

  1. 编程基础与基本工具的掌握
  2. 算法与数据结构达到合格标准
  3. 有自己感兴趣的方向，知道对应岗位基本行情
  4. 岗位需要的专业知识过硬，生产环境的工具有一定理解

由于这几项内容是相互交织的，与其严格区分学习阶段，不如尽快开始全面学习、相互促进，能达到更好的效果。

因此，基本的学习进度建议如下：

  1. 在前3个月，初步确定自己感兴趣的方向与目标岗位
  2. 在前2个月，搞定《算法导论》，要学习与刷题相结合
  3. 在随后的半年，学专业课知识，尤其重点学习岗位相关的内容；此阶段同时保持每天1~2小时刷算法题
  4. 最后4个月，利用前面的学习成果和笔记，对一线生产环境的实际内容进行学习和思考，并写好简历准备笔试面试

计划问答摘要

  1. 怎么初步确定自己感兴趣的方向与目标岗位，牛客网吗？
    • 主要是找学长了解，学校请学长来做的分享要积极去听，自己也要积极主动接触。
  2. 算法公开课、看书、刷题，是这个流程吗，题去哪里刷呢？
    • 我的公开课是对书籍的一个导读和补充，两个你一起看
    • 刷题的话，除了书上例题，还可以去找找网上OJ的题，你可以多去知乎、百度、谷歌查查每个算法、数据结构对应哪些题，很好查到的；每个算法、数据结构都得刷上至少一两道，才好理解掌握
    • 而且提醒一下，最开始一定要先看我公开课的前两节，尤其是第二节的纸笔模拟方法，一定要掌握并坚持使用，否则很难做到深刻理解
  3. 最后4个月那些，是指实习吗？
    • 那个时间还没到秋招，所以并非实习，而是去主动了解。各岗位描述中都写着需要掌握的技术，对应去学习就好了
  4. 学专业课知识，尤其重点学习岗位相关的内容。不同岗位器重的专业课有很多区别吗？
    • 是的，比如你搞底层，对操作系统这门课要求就很高；如果你搞后端，那计算机网络就肯定要学好
    • 不同岗位要求还是有明显差异的，所以不必每门课都学得非常优秀，都要会，但要有所选择和精通
  5. 我们学校假期会有实训安排，就是Java开发的技术栈，大二到大四不断加深，如果把这个作为技术方向可以吗？
    • 可以，但你还是要充分了解再决定，而且就算Java开发也有不同方向，你没有方向倾向就不知道学习重点。你就算目前按照学校的安排，工作之后在方向上还是要有所侧重的~
