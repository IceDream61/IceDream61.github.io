教学计划

学习主线是C/C++，语言与相关知识各占一半。

  1. HelloWorld：安装CLion + 中文插件，运行C语言HelloWorld

      • 讲解IDE，演示使用、自己搭建
      • 讲解HelloWorld，先屏蔽底层原理、只讲运行逻辑
      • 讲解c语言基础，演示为主、其余部分留作自学
    2. 自学：学习c语言基础 by 《C 语言教程 | 菜鸟教程》
      • C 基本语法
      • C 数据类型
      • C 变量
      • C 常量
      • C 存储类（选）
      • C 运算符
      • C 判断
      • C 循环
      • C 输入 & 输出（with “C 常量”中的“字符常量 - 转义序列”部分）
      • 《算法大讲堂 02 充分理解靠纸笔 | 哔哩哔哩》
    3. 作业
      • 运行HelloWorld：搭建好学习环境
      • 斐波那契数列、打印菱形、猜数游戏：学会基础语法
      • 预习纸笔模拟方法
  2. 代码逻辑与算法：理解代码运行逻辑、纸笔模拟方法、调试方法，了解“算法”概念
    1. 课堂：zoom分享屏幕，时间各半
      • 解决上堂课的所有问题 at c语言基础
      • 解决疑问 at 《算法大讲堂 02 充分理解靠纸笔 | 哔哩哔哩》
      • 进一步学习调试
      • 了解什么是“递归”
      • 了解什么是“算法”
    2. 自学：继续学习c语言 by 《C 语言教程 | 菜鸟教程》
      • C 函数
      • C 作用域规则
      • C 数组
      • C enum（枚举）
      • C 递归
