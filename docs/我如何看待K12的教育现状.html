目前的K12教育现状，其实是在畸形的道路上越行越远。我无意参与什么辩论，但我觉得有必要把自己的思考放出来，和有兴趣的人一起交流。

从国家社会的角度来说，现有教育体系是为了国家工业化服务的。之所以这么说，原因如下：

  1. 根据官方xxx的数据，中国目前从事农业生产的人数占全国人数的xx%，而从事工业生产的人数只占xx%。
  2. 很显然，如此一来，让全民接受九年义务教育，从个人的生产生活来说，是没有意义的。也就是说，所谓的提高全民素质，全称应该是提高全民工业化素质。
  3. 故而，若非为了国家工业化有足够的劳动者，完全可以让学生按比例接受农业或工业教育体系，而非像目前这般所有人以高考为衡量标准。
  4. 可用的劳动者有三个必要条件：劳动意愿、劳动纪律、劳动技能。K12阶段，培养的主要是劳动纪律，劳动技能是大学培养的内容，劳动意愿则是无需培养的。

综上，我做出的判断是：K12阶段只是为了培养大部分学生的劳动纪律，同时为少量高端人才培养好进一步深造的劳动技能基础。

如此看来，K12阶段家长们对学生的大力培养，严苛点说就是飞蛾扑火。为什么呢？是几个原因综合作用的结果：

  1. 学生只要接受到合格的教育，就足够让有潜力的孩子脱颖而出，进入合适的成长轨道。因此，家长应该做的是尽量让孩子走上合适的路径，而非单纯砸钱拼分。
  2. 教育体系自从推向市场，老师们就主要凭着良心师德的惯性来教书育人，惯性总会消耗殆尽，因此客观上孩子的教育成本只能越来越高。
  3. 全民的独生子女现状导致家长对孩子教育期望崎高，而大部分学生家长的信息和能力对孩子的教育需求来讲，是永远也不够的。
  4. 这时候，教育产业的繁荣，本质上是满足了家长有钱有心无力导致的需求，既解决了家长为孩子寻个好前途的简单愿望，又解决了家长和学生的焦虑。
