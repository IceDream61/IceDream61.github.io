计算机科学与技术 笔记

基础工具

  • 在计算机的世界中，日常用到的工具数不胜数。而且不客气地说，我们还处于计算机行业的石器时代，工具还处于广泛尝试且标准不一的时期。
  • 因此，每个阶段的coder都需要一份 工具指南 ，结合当下需求，来指导自己如何选择、扩展乃至开发一个适合自己的工具。
  • 我很希望做出这样一份指南，但我知道这是一个浩大的工程，或许将来有条件了可以去做，但现阶段绝非我一个人可以完成的。我所能做的第一步，就是把自己对工具的见识和理解都先记录、总结、分享出来，如果有人关注的话也结合一些意见，从而慢慢整理出一份我所涉及领域的工具指南，还有一套工具探索的思路。

社区个人

  • 平时遇到问题，第一反应都是Google来解决。说是Google，实际上背后是各大社区所带来的讨论、文章、书籍、站点。
  • 大部分散乱的内容只需要记录一个链接即可，而很多优质内容整合的站点则有记录下来的必要，想了解特定领域的时候去看看，非常受益。
  • 还有很多这个行业相关的讨论，感兴趣的话也可以常去看看。

书籍课程

  • 计算机领域的书籍很多，课程更是鱼龙混杂，因此很有整理的必要。
  • 尤其是很多优质的出版社、大神，还有很多优秀的单品书籍，都是宝贵的学习资料，不容错过。
  • 还有很多这个行业相关的思考，我这种业内人员还是很关心的。
