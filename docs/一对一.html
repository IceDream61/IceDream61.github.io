一对一教学

一对一的性价比是最高的，也是最能因材施教的。

教学方式

  1. 线上线下都可以，线下效果会好很多，但需要来杭州。
  2. 一次课最好是2小时课程 + 6小时辅导，哪怕没有这个条件，至少也要有2小时的专心上课时间。
  3. 很多看书就能懂的内容并没有讲授的必要，因此课上只讲书上学不到的内容，学生必须保质保量完成作业。
  4. 前期交流和试听课只能确定基本教学计划，具体教学进度需要根据学生情况实时调整，因此能学多快取决于学生的天赋和努力。
  5. 课程中交流的占比可能会很高，不要担心，放轻松，我会引导你的思考。

招收标准

  1. 有兴趣。
  2. 有时间。
  3. 积极思考。
  4. 认真完成作业。
